# 第一章 Python缩进
缩进指的是代码行开头的空格。
Python使用缩进来指示代码块。
如果省略缩进Python会出错；
实例：
语法错误：
```
if 5 > 2
    print("FIve is greater than two")
# 如果省略缩进，Python会出错。

```
