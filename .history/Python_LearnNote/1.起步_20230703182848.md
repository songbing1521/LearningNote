# 第一章 Python缩进

缩进指的是代码行开头的空格。
Python使用缩进来指示代码块。
如果省略缩进Python会出错；
实例：
语法错误：

```py
if 5 > 2
    print("FIve is greater than two")
# 如果省略缩进，Python会出错。

```

## Python变量

在程序设计中，变量是一种存储数据的载体。计算机中的变量是实际上存在的数据或者说是存储器中存储数据的一块内存空间。变量的值可以被读取和修改，这是所有计算和控制的基础。
在Python中， 变量是在为其赋值时创建的：
实例：
Python中的变量：

```python
x = 5
y = "hello,world!"
```

Python中没有声明变量的命令。
