# 算法分析

算法是为求解一个问题需要遵循的、被清楚指定的简单指令的集合。算法分析就是对算法的研究，是指对算法的优劣程度进行评估。算法分析的目的是为了选择合适的算法，或者改进算法的设计。在本章笔记将要讨论的是：
+ 如何估计一个程序所需要的时间
+ 如何将一个程序的运行时间降低
+ 粗心地使用递归的后果
+ 用于将一个数自乘得到其幂以及计算两个数最大公因数的非常有效的算法
